# Question
Show an easier way to write this array:

```ruby
flintstones = ["Fred", "Barney", "Wilma", "Betty", "BamBam", "Pebbles"]
```

# Solution
```rb
flintstones = %w(Fred Barney Wilma Betty BamBam Pebbles)
```
